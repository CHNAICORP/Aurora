## 🌟 项目简介

Aurora是一个功能丰富的全栈个人博客系统，采用星空主题设计，支持文章发布、视频播放、图片展示等多种内容形式。系统涵盖技术开发、国学文化、股市投资、虚拟货币等多个领域，旨在打造一个知识分享与思想交流的平台。

## ✨ 主要特性

- **🎨 星空主题设计** - 独特的视觉体验，支持深色/浅色模式
- **🌐 国际化支持** - 支持中文、英文、日文、韩文多语言切换
- **📝 富文本编辑** - 强大的文章编辑功能，支持图片、视频嵌入
- **🎥 多媒体支持** - 视频播放、图片展示、文件管理
- **💬 社交互动** - 评论系统、点赞、收藏、关注功能
- **👤 用户系统** - 完整的注册登录、个人中心、权限管理
- **📱 响应式设计** - 完美适配桌面端、平板、手机等设备
- **🔍 搜索功能** - 支持全文搜索，快速找到感兴趣的内容

## 🛠️ 技术栈

### 前端技术
- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Element Plus** - Vue3组件库
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Axios** - HTTP客户端

### 后端技术
- **Node.js** - JavaScript运行时
- **Express** - Web应用框架
- **MongoDB** - NoSQL数据库
- **Mongoose** - MongoDB对象建模
- **Redis** - 缓存数据库
- **Socket.IO** - 实时通信

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0
- MongoDB >= 4.0
- Redis >= 6.0 (可选)
- npm >= 6.0

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/yourusername/aurora.git
cd aurora
```

2. **启动后端服务**
```bash
cd aurora-backend
npm install
npm run dev
```

3. **启动前端应用**
```bash
cd aurora-frontend
npm install
npm run dev
```

4. **访问应用**
- 前端：http://localhost:5173
- 后端API：http://localhost:3000
